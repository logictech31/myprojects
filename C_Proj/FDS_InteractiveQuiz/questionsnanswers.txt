
Basic

1. Who is father of C Language?
A. Dennis Ritchie
B. Bjarne Stroustrup
C. Dr. E.F. Codd
D. James A. Gosling
Answer : A

2. C programs are converted into machine language with the help of 
A. An Editor
B. A compiler
C. An operating system
D. None of the above
Answer : B

3. A C variable cannot start with
A. An alphabet
B. A number
C. A special symbol other than underscore
D. both (b) and (c)
Answer : D

4. Which of the following is allowed in an Arithmetic instruction in C?
A. []
B. {}
C. ()
D. None of the above
Answer : C

5. What type of language is C?
A. Object Oriented
B. Procedural
C. Object Based
D. Recursive
Answer : B


Medium 

1. Array elements are always stored in _________ memory locations.
A. Sequential
B. Random
C. Sequential and Random
D. None of the above
Answer : A

2. What is the right way to access value of struct book{ int price, page; }?
A. printf(“%d%d”, book.price, book.page);
B. printf(“%d%d”, price.book, page.book);
C. printf(“%d%d”, price::book, page::book);
D. printf(“%d%d”, price->book, page->book);
Answer : A

3. perror( ) function used to ?
A. Work same as printf()
B. prints the error message specified by the compiler
C. prints the garbage value assigned by the compiler
D. None of the above
Answer : B

4. A binary tree with 27 nodes has _______ null branches.
A. 54
B. 27
C. 26
D. None of the above
Answer : D

5. What will printf(“%d”, 10 ? 0 ? 5 : 1 : 12); print on screen?
A. 10
B. 0
C. 12
D. 1
Answer : D
